# BattleShipGame
## ルール

1. 10*10のマスに艦船を自由配置(艦船の種類は後述)
2. 自分のターンに攻撃するマスを指定
3. 相手はあたったかどうか宣言、撃沈したらその旨も伝える
4. あたっていたら続けて攻撃できる
5. 外したら相手のターンへ

艦船の種類

* 空母：5マス
* 戦艦：4マス
* 巡洋艦(今回は2隻使用)：3マス
* 駆逐艦：2マス

縦においても横においてもオッケー
その船のすべてのマスが当てられたら撃沈

## 探索アルゴリズム

置ける場所カウント方式を採用
3*3で駆逐艦(2マス)を置くときを考える
この時、置いてある確率はすべての置き方を考慮すると
|2||3|2|
|3||4|3|
|2||3|2|
のように分布されると考えられる。
よって真ん中の4を選択するのが適切であると考える。
